# Preview 预览

#### 使用

```html
<!--基本使用-->
<h-preview :url="link" small/>
```

#### 属性

| 属性名     | 类型      | 说明                    | 默认值   |
|:--------|---------|-----------------------|:------|
| url     | string  | 要预览的文件链接（目前仅支持图片和pdf） |       |
| urlList | array   | 多文件预览                 |       |
| small   | boolean | 预览组件大小                | false |
