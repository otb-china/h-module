# 样式类

### 基本

```html
<!--边距置空-->
<p class="back-gauge">内容</p>

<!--居中-->
<div class="align-center"></div>

<!--左对其-->
<div class="align-left"></div>

<!--右对其-->
<div class="align-right"></div>

<!--手手光标-->
<p class="pointer"></p>

<!--表格容器（阴影 表格最小高度 标签相邻间隔）-->
<div class="table-container"></div>

<!--按钮容器-->
<div class="button-container"></div>
```

### link

```html
<!--跳转文字悬停动画-->
<span class="link">跳转内容</span>

<!--较长的跳转文字悬停动画-->
<span class="long-link">跳转内容</span>
```

### animation

```html
<!--呼吸动画-->
<div class="breathe">呼吸内容</div>

<!--闪烁动画-->
<div class="flashing">闪烁内容</div>

<!--跳动动画 元素波浪跳动 盒子 flex-->
<div style="display: flex">
  <span class="beating">测</span>
  <span class="beating">试</span>
  <span class="beating">内</span>
  <span class="beating">容</span>
</div>
```

### flex

```html
<!--不换行-->
<div class="flex-row"></div>

<!--自动换行-->
<div class="flew-warp"></div>

<!--居中-->
<div class="flex-center"></div>

<!--元素均分-->
<div class="space-around"></div>

<!--间隔均分-->
<div class="space-evenly"></div>

<!--两端对齐-->
<div class="space-between"></div>
```

### font

```html
<!--粗体-->
<p class="bolder">内容</p>

<!--字体大小 FS(1-50)-->
<p class="FS20">内容</p>

<!--文字不换行-->
<p class="nowrap">内容</p>
```

### margin

```html
<!--上边距 MT(1-50)-->
<div class="MT20"></div>

<!--右边距 MR(1-50)-->
<div class="MR20"></div>

<!--下边距 MB(1-50)-->
<div class="MB20"></div>

<!--左边距 ML(1-50)-->
<div class="ML20"></div>

<!--上下边距 MY(1-50)-->
<div class="MY20"></div>

<!--左右边距 MX(1-50)-->
<div class="MX20"></div>

<!--外边距 M(1-50)-->
<div class="M20"></div>
```

### padding

```html
<!--上边距 PT(1-50)-->
<div class="PT20"></div>

<!--右边距 PR(1-50)-->
<div class="PR20"></div>

<!--下边距 PB(1-50)-->
<div class="PB20"></div>

<!--左边距 PL(1-50)-->
<div class="PL20"></div>

<!--上下边距 PY(1-50)-->
<div class="PY20"></div>

<!--左右边距 PX(1-50)-->
<div class="PX20"></div>

<!--内边距 P(1-50)-->
<div class="P20"></div>
```
